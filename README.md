# Paradise Nursery Shopping Application

## Project Overview
This project is a dynamic web application built with React and Redux for an online plant shop called Paradise Nursery. It allows users to browse plants, view details, add items to a shopping cart, manage cart quantities, and calculate totals.

## Features
- Landing page with company name and "Get Started" button.
- Product listing page with categorized houseplants, thumbnails, names, prices, and "Add to Cart" functionality.
- Shopping cart page with item management (increase/decrease quantity, delete), total costs, and navigation.
- Navigation bar on product and cart pages.
- Redux for state management of the shopping cart.

## Technologies Used
- React
- React Router DOM
- Redux Toolkit
- CSS for styling

## Setup Instructions
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm start` to start the development server.
